# 벙커마켓 · 캔바(Canva) 카드뉴스 작업 킷

캔바에서 카드뉴스 만들 때 이 문서 하나만 옆에 두면 됨.
(무료 계정으로 충분. 색은 수동 입력하면 됨)

---

## 1. 템플릿 고르기 (여기서 90% 결정됨)

캔바 검색창에 아래 순서로 검색:

1. **`minimal carousel`** (영어 검색이 템플릿 질이 훨씬 좋음)
2. `photo carousel minimal`
3. 한글로는 `카드뉴스 미니멀` (단, 한글 템플릿은 장식 과한 게 많음 — 주의)

**고르는 기준 — 이 4개 다 만족해야 함:**
- [ ] 6페이지 이상 구성
- [ ] 제목이 화면의 1/3 이상을 차지 (큰 타이포)
- [ ] 사진 넣는 자리가 있음
- [ ] 장식 요소(스티커·이모지·프레임)가 거의 없음

**거르는 기준:** 그라디언트 배경, 이모지 장식, 3가지 이상 색 쓰는 템플릿
→ 지금 우리가 벗어나려는 바로 그 느낌임.

---

## 2. 브랜드 컬러 (색상 코드 직접 입력)

| 용도 | HEX | 설명 |
|---|---|---|
| 배경(짙은) | `#08283A` | 깊은 바다 — 표지·마지막 장 |
| 배경(밝은) | `#EEF6F5` | 포말 — 본문 장 (밝/어둡 교차시킬 것) |
| 포인트 | `#E8654F` | 산호 — 숫자·강조·CTA 딱 한 군데씩만 |
| 보조 | `#1C7A86` | 조수 청록 — 작은 라벨·키커 |
| 모래(따뜻한 강조) | `#E8D9BD` | 마무리 문장 한 줄 정도 |

**규칙: 한 장에 색 3개 이상 쓰지 말 것.** (배경 1 + 글자 1 + 포인트 1)

---

## 3. 폰트 (캔바에 있는 것 중에서)

| 용도 | 폰트 | 크기 감각 |
|---|---|---|
| 대제목 | **나눔명조** (Nanum Myeongjo) | 화면 폭의 80% 차지할 만큼 크게 |
| 본문·설명 | **프리텐다드** (Pretendard) 또는 나눔고딕 | 대제목의 1/6 이하로 작게 |
| 숫자 강조 | 프리텐다드 ExtraBold | "0원"은 화면 절반 크기로 |

**규칙: 폰트 2개만. 크기 대비는 소심하게 말고 10배로.**
(사이트 제목이 명조 계열이라 나눔명조가 브랜드와 맞음)

## 4. 장별 붙여넣기 문구 (복사용)

### 카드 1 — 표지 (어두운 배경 #08283A)
```
농부님께 드리는 제안

바다 소리와 함께,
농부님의 농산물을
소개하고 싶습니다

벙커마켓
```

### 카드 2 — 소개 (밝은 배경 #EEF6F5)
```
벙커마켓이 뭐 하는 곳인가요?

바닷가의 소리를 녹음해
작은 카드에 담습니다.

손님이 카드에 휴대폰을 대면 —
그 바다의 파도 소리가 흐르고,
그 바닷가에서 자란 농산물이 함께 소개됩니다.
```

### 카드 3 — 손님의 30초 (밝은 배경, 숫자는 #E8654F)
```
손님이 겪는 30초

1  장터에서 카드에 폰을 댄다
2  파도 소리가 흐른다
3  "이 바닷가의 당근이구나"
4  [사러 가기] → 농부님 판매처로

손님은 소리로 산지를 먼저 만납니다.
```

### 카드 4 — 하실 일 (어두운 배경)
```
농부님은 무엇을 하나요?

링크 하나만
주시면 됩니다

· 팔던 방식 그대로 파세요 (네이버 스토어 · 전화 주문 · 직거래)
· 저희 '사러 가기' 버튼이 농부님 판매처로 바로 연결됩니다
· 재고 · 배송 · 정산, 저희는 손대지 않아요
```

### 카드 5 — 조건 (밝은 배경, "0원"을 화면 절반 크기로)
```
조건은요?

입점비 0원
지금은 수수료 0원
과장 없음

"소리가 흐르는 곳에서 자란" — 사실만 말합니다.
저희가 얻는 건 '산지의 이야기'입니다.
```

### 카드 6 — 연락 (어두운 배경, 이메일은 크게)
```
함께하려면, 세 가지만 알려주세요

1  농가(농부님) 이름
2  팔고 있는 것
3  판매처 링크 또는 전화번호

play@soundb.kr
마르셰 장터에서 직접 만나실 수도 있어요
```

## 5. 프로 느낌 내는 규칙 5개 (템플릿 고르고 나서)

1. **한 장 = 한 메시지.** 문구가 템플릿 칸보다 많으면 문구를 지울 것 (칸을 늘리지 말고).
2. **밝/어둡 교차.** 표지 어둡게 → 2·3·5 밝게 → 4·6 어둡게. 넘길 맛이 생김.
3. **사진 슬롯엔 진짜 바다·밭 사진.** 없으면 Canva 무료 스톡에서 `jeju sea`, `korean farm` 검색 (단, 실제 산지 사진 생기면 즉시 교체 — "진짜"가 이 브랜드의 전부).
4. **포인트색(#E8654F)은 장당 1곳.** 숫자든 단어 하나든. 두 곳부터는 싸구려 됨.
5. **로고 자리 통일.** 모든 장 같은 위치에 작게 "● 벙커마켓" (원은 #E8654F).
